# Image Filter Application

A Python-based image processing application that applies multiple visual filters to images using the Pillow library.

## Features

- Grayscale filter
- Blur effect
- Color inversion
- Image file processing
- Automatic filtered image export

## Technologies Used

- Python 3
- Pillow (PIL)

## Filters Included

- Grayscale conversion
- Blur filter
- Inverted colors

## Output Files

The application generates:

- grayscale.jpg
- blurred.jpg
- inverted.jpg

## Concepts Demonstrated

- Image processing
- File handling
- External Python libraries
- Image manipulation techniques
